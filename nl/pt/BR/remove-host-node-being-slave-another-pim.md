# Removendo um nó do host no PIM
{: #removing-a-host-node-in-pim}

Conclua as etapas a seguir para remover um nó do host como um escravo no PIM:

1. Acesse **Infraestrutura > Nós de hardware**.
2. Selecione o nó que você deseja desassociar.
3. Clique em **Cancelar registro**.
4. Clique em **OK**.
