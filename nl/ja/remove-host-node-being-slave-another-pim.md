# PIM でのホスト・ノードの削除
{: #removing-a-host-node-in-pim}

PIM で次の手順を実行すると、ホスト・ノードのスレーブの設定を解除できます。

1. **「インフラストラクチャ」>「ハードウェア・ノード (Hardware Nodes)」**に移動します。
2. 関連付けを解除するノードを選択します。
3. **「登録解除 (Unregister)」**をクリックします。
4. **「OK」**をクリックします。
