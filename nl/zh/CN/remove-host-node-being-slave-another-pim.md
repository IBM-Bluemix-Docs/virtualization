# 除去 PIM 中的主机节点
{: #removing-a-host-node-in-pim}

完成以下步骤来除去 PIM 中作为从属节点的主机节点：

1. 转至**基础架构 > 硬件节点**。
2. 选择要解除关联的节点。
3. 单击**注销**。
4. 单击**确定**。
