# 在 PIM 中添加主机节点
{: #adding-a-host-node-in-pim}

完成以下步骤在 PIM 中将主机节点添加为其他节点的从属项：

1. 在指定为“主节点”的服务器上，转至**基础架构 > 硬件节点 > 新建硬件节点**。
2. 输入远程硬件节点的 IP 地址/用户名 (root)/密码并进行注册。

新的节点资源不会立即在主 PIM 中进行更新。
{:tip}
