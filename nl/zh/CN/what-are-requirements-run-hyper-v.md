# Hyper-V：硬件需求
{: #hyper-v-hardware-requirements}

Hyper-V 必须安装在 Windows 2008 Server Datacenter Edition（64 位）上。所有 {{site.data.keyword.cloud}} 硬件均满足 Hyper-V 的系统需求。Microsoft 建议服务器上至少有 2 GB RAM。
