# 在 PIM 中移除主機節點
{: #removing-a-host-node-in-pim}

請完成下列步驟，在 PIM 中移除主機節點以作為從屬節點：

1. 移至**基礎架構 > 硬體節點**。
2. 選取您要取消關聯的節點。
3. 按一下**取消登錄**。
4. 按一下**確定**。
