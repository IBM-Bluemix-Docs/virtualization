# 訂購 Hyper-V
{: #ordering-hyper-v}

如果您有現有的伺服器，則可以在伺服器上安裝 Hyper-V，方法為起始作業系統重新載入，並將作業系統變更為 `Windows 2008 Server Datacenter Edition（含 Hyper-V）`。

如果您想要使用 Hyper-V 來佈建伺服器，請訂購新的伺服器，並選取 `Windows 2008 Server Datacenter Edition（含 Hyper-V）` 作為作業系統。
