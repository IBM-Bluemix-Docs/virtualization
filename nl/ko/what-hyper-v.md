# Hyper-V의 개념
{: #what-is-hyper-v-}

Hyper-V는 Windows 2008 Server Datacenter Edition의 가상화 시스템으로, 단일 물리적 서버가 자체 운영 체제를 모두 실행 중인 여러 가상 머신을 호스팅할 수 있도록 해줍니다.
