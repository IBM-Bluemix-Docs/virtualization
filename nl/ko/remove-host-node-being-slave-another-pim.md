# PIM에서 호스트 노드 제거
{: #removing-a-host-node-in-pim}

호스트 노드를 PIM에서 슬레이브로 제거하려면 다음 단계를 완료하십시오. 

1. **인프라 > 하드웨어 노드**로 이동하십시오.
2. 연결 해제하려는 노드를 선택하십시오. 
3. **등록 취소**를 클릭하십시오.
4. **확인**을 클릭하십시오.
