# Hyper-V: 하드웨어 요구사항
{: #hyper-v-hardware-requirements}

Hyper-V는 Windows 2008 Server 64비트 Datacenter Edition에 설치해야 합니다. 모든 {{site.data.keyword.cloud}} 하드웨어는 Hyper-V의 시스템 요구사항을 충족합니다. Microsoft는 서버에 최소 2GB의 RAM을 권장합니다.
